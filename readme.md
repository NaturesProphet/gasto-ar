# Clima-App

Descubra qual a média de gasto mensal do seu ar condicionado

## Instalação

```shell
$ npm install -g gasto-ar
```

## Modo de usar

```shell
$ gasto-ar <potencia-ar-condicionado> <horas-de-uso-diario>
$ gasto-ar 680 8
```
